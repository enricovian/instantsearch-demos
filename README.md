## About this demo

Custom widget to apply arbitrary search configurations.
For example, buttons to apply filters with a click.

## Get started

To run this project locally, install the dependencies and run the local server:

```sh
npm install
npm start
```

Alternatively, you may use [Yarn](https://http://yarnpkg.com/):

```sh
yarn
yarn start
```
